# 서 울 특 별 시

이 문서는 내부결재용으로 작성되었으며, 수신자, 경유자, 제목 항목으로 구성되어 있습니다. 수신자는 내부결재이며, 경유자는 명시되어 있지 않습니다. 제목은 "[민원답변] 청년수당과 국민취업지원제도 참여"입니다.

안녕하십니까?

귀하께서 응답소(국민신문고)를 통해 신청하신 민원(접수번호: 20240906901012, 국민신문고번호: 1AA-2409********)에 대한 검토 결과를 다음과 같이 알려드립니다.

귀하의 민원내용은 "국민취업지원제도 1유형(구직촉진수당) 수급 이후 청년수당 신청 가능 여부"에 관한 것으로 판단 됩니다.

귀하의 질의사항에 대해 검토한 의견은 다음과 같습니다.

앞선 민원에서 안내드렸듯이, 서울 청년수당 사업은 고용노동부 「국민취업지원제도」 1유형 참여자에 대하여 중복 사업 참여를 제한하고 있습니다. 국민취업지원제도는 구직촉진수당을 월 50만원씩 6개월로 지급하는 사업으로서, 청년수당과 유사한 사업으로 인정되기 때문입니다.

따라서, 국민취업지원제도 1유형과 청년수당을 동시에 참여할 수는 없지만, 국민취업지원제도(1유형)의 구직촉진수당을 "청년수당 모집공고 전월까지" 수급 완료 시, 청년수당 사업에 참여가 가능합니다. 예를 들어, 청년수당 모집일이 6월 10일인 경우, 5월까지 국민취업지원제도 1유형을 수급 완료하셨을 경우 청년수당 사업에 참여가 가능합니다.
