# 서 울 특 별 시

이 문서는 안심소득과장님께 발송된 공문입니다. 제목은 "[회신] 안심소득 지원대상자 '서울 청년수당' 6월분 지급여부 확인 요청"입니다.

본문은 다음과 같은 내용으로 구성되어 있습니다.

첫째, 서울특별시 안심소득추진과-5083호(2024년 6월 19일 발송, "안심소득 지원대상자 '서울 청년수당' 지원여부 확인 요청")와 관련된 회신입니다.
